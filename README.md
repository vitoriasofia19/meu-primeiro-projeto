# Meu Primeiro Projeto

Um projeto simples em Python para praticar o fluxo de contribuicao no GitHub.

## Sobre

Este repositorio contem uma pequena calculadora de linha de comando, criada
para fins de aprendizado do fluxo de Pull Requests.

## Como usar

```bash
python calculadora.py
```

## Funcionalidades

- Soma
- Subtracao
- Multiplicacao
- Divisao

## Testes

```bash
python -m unittest discover -s tests -t .
```
